# Yahtzee Game

A graphical Yahtzee game implementation using Python and pygame.

## Setup

This project uses [uv](https://github.com/astral-sh/uv) for dependency management.

### Prerequisites
- Python 3.10 or higher
- uv package manager

### Installation

1. Clone the repository
2. Install dependencies:
```bash
uv sync
```

### Running the Game

```bash
uv run python main.py
```

## Game Rules

Yahtzee is a dice game where players roll 5 dice up to 3 times per turn, trying to achieve specific combinations to score points across 13 categories.
